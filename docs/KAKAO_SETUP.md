# 카카오 API 설정 가이드

## 카카오 맵 API 키 발급

### 단계별 가이드

1. **카카오 개발자 센터 접속**
   - https://developers.kakao.com/ 접속
   - 카카오 계정으로 로그인

2. **애플리케이션 등록**
   - 우측 상단 "내 애플리케이션" 클릭
   - "애플리케이션 추가하기" 클릭
   - 앱 이름 입력 후 저장

3. **JavaScript 키 확인**
   - 등록한 애플리케이션 선택
   - 좌측 메뉴 "앱 키" 클릭
   - **JavaScript 키** 복사

4. **플랫폼 설정**
   - "플랫폼" 메뉴 클릭
   - "Web 플랫폼 등록" 클릭
   - 사이트 도메인: `http://localhost:3000` (개발용)

## 카카오 로그인 설정

### Redirect URI 설정 위치

1. **카카오 개발자 센터 접속**
   - https://developers.kakao.com/ 접속
   - 등록한 애플리케이션 선택

2. **카카오 로그인 메뉴로 이동**
   - 좌측 메뉴에서 **"제품 설정"** 클릭
   - **"카카오 로그인"** 클릭

3. **Redirect URI 찾기**
   - 페이지 중간 또는 하단에 **"Redirect URI"** 섹션이 있습니다
   - 스크롤해서 찾으세요

4. **Redirect URI 추가**
   - "Redirect URI" 섹션에서 **"URI 추가"** 버튼 클릭
   - 입력창에 다음 URI 입력:
     ```
     http://localhost:3000/auth/kakao/callback
     ```
   - **"저장"** 버튼 클릭

### 스크린샷 위치 안내

카카오 개발자 센터 구조:
```
내 애플리케이션
├── 앱 설정
│   ├── 앱 키
│   └── 플랫폼
└── 제품 설정  ← 여기!
    ├── 카카오 로그인  ← 여기 클릭!
    │   ├── 활성화 설정
    │   ├── 동의항목
    │   └── Redirect URI  ← 여기서 설정!
    ├── 카카오톡 메시지
    └── 기타 제품들...
```

### 주의사항

- Redirect URI는 정확히 일치해야 합니다
- `http://localhost:3000/auth/kakao/callback` (O)
- `http://localhost:3000/auth/kakao/callback/` (X - 마지막 슬래시 주의!)
- 개발용과 운영용을 각각 등록할 수 있습니다

### 문제 해결

**Q: Redirect URI를 찾을 수 없어요**
- "제품 설정" → "카카오 로그인" 메뉴로 들어가세요
- "활성화 설정"을 먼저 ON으로 설정해야 보일 수 있습니다

**Q: "Redirect URI가 일치하지 않습니다" 에러**
- 등록한 URI와 실제 사용하는 URI가 정확히 일치하는지 확인
- 프로토콜(http/https), 도메인, 경로 모두 확인
- 마지막 슬래시(/) 유무도 중요합니다
